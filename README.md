**文件使用方法：**

如果没有vue-cli全局安装

`npm install -g vue-cli`


也可以自行初始化项目后将src文件替换掉

`vue install webpack 你的项目名称`

`cd 你的项目名称`

安装依赖

`npm install`

本地启动

`npm run dev`
